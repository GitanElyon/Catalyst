# Catalyst

Catalyst is a free lightweight IDE designed to expidite the process of idea to product.

## Purpose

I have been plauged for the longest time by IDEs that claim to be lightwieght, then take up all my RAM (I'm looking at you VSC). And while there are many lightweight options, none of them really have the feature pool I am looking for, so I set out to build my own.

Catalyst aims to solve all of these problems by providing a clean, lightweight developing expirence.

## Some History

Recently, I have been daily driving Arch Linux, and that means two things. The first is that I have learned a lot about computers on the low level.

But I have also discovered the beauity of tiled window managers. I love the idea of a tiled window manager, but I hate that editors like VSC waste so much of my screen space with icons and buttons.

With a need for a new look in my life, as well as a light weight editor, I set out to build Catalyst.
